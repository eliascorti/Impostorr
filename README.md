# Impostorr

Juego de fiesta web para jugar al impostor pasando el dispositivo entre los jugadores. Permite seleccionar entre más de 25 categorías con al menos 30 palabras cada una.

## Cómo usarlo

1. Abre `index.html` en tu navegador favorito.
2. Define la cantidad de jugadores (mínimo 3) y elige una de las categorías disponibles.
3. Opcionalmente selecciona el modo difícil para jugar con dos palabras similares.
4. Presiona **Comenzar ronda** y pasa el dispositivo para que cada jugador revele su palabra o si es el impostor.
5. Después de que todos vean su rol, comiencen la discusión y voten al impostor.

Puedes reiniciar la partida en cualquier momento con el botón **Nueva ronda**.
